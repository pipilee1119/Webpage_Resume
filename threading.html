Python多線呈操作

紀錄學習threading使用筆記。

範例程式

                        import threading
import time
from queue import Queue
def job(l,q):
    for i in range (len(l)):
        l[i] = l[i]**2
    q.put(l)

def multithreading():
    q = Queue()
    threads = []
    data = [[1,2,3],[3,4,5],[4,4,4],[5,5,5]]
    for i in range(4):
        t = threading.Thread(target=job,args=(data[i],q))
        t.start()
        threads.append(t)
    for thread in threads:
        thread.join()
    results = []
    for _ in range(4):
        results.append(q.get())
    print(results)
multithreading()

主要使用

                        
t = threading.Thread(target=job,args=(data[i],q))
t.start()

建立thread:   threading.Thread(target = "要執行的function",args = "該func的輸入參數")
開始運行該thread:  t.start

                        
threads = []
threads.append(t)
for thread in threads:
    thread.join()

建立thread陣列用於後續join
t加入陣列後，for迴圈確認陣列內的thread都已執行完畢，才進入下段程式(防止執行混亂)。
