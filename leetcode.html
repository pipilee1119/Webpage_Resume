1.Two Sum

從一個陣列(nums)中找出兩個加總等於target的數。

Example.1

                        Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Output: Because nums[0] + nums[1] == 9, we return [0, 1].

雙for迴圈

最簡易的作法: 雙for迴圈，時間複雜度O(n^2)。

                        for i in range(len(nums)):
        for j in range(i,len(nums)):
             if nums[i]+nums[j]==target:
                   return [i,j]
    return []

使用hashmap做紀錄

Python使用dictionary紀錄target - 陣列中的每個數，例如[1,2,5]；target = 7，dict紀錄: 7-1 ,7-2 => [6:0,5:1] ，for迴圈到最後一個數5的時候，在dict中找到5，則回傳nums位置[1,2]，時間複雜度O(n)。

                        ht = {}
for i,num in enumerate(nums):
    if target-num in ht:
        return [ht[target-num],i]
    ht[num] = i
return []
